
# Stock API Project

## 📌 Description
This project is a Spring Boot REST API to manage stock market data. It allows users to perform CRUD operations and handle stock information efficiently.

## ⚙️ Tech Stack
- Java
- Spring Boot
- MySQL / H2 Database
- REST API
- Git & GitHub

## 🚀 Features
- Add new stock
- View all stocks
- Update stock details
- Delete stock
- RESTful API design

## 📡 API Endpoints
- GET /stocks → Get all stocks  
- GET /stocks/{id} → Get stock by ID  
- POST /stocks → Add new stock  
- PUT /stocks/{id} → Update stock  
- DELETE /stocks/{id} → Delete stock
